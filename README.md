# Clarity - Chrome Extension (Frontend)

Clarity is a Chrome Extension designed to make the web more accessible. It extracts complex text from websites (like Wikipedia) and sends it to a dedicated AI backend for simplification.

## 🔗 Related Repositories
* **AI Backend Engine:** [clarity-backend](https://github.com/welbornecuthbert-dotcom/clarity-backend)

## 🎨 Key Features
* **Status Overlays:** Real-time feedback while the AI is thinking.
* **Auto-Scroll:** Automatically brings the simplified text into view.
* **Smart Extraction:** Only grabs relevant article content, ignoring sidebars and ads.

## 🛠️ Installation
1. Clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select this folder.
5. **Note:** Requires the [Clarity Backend](https://github.com/welbornecuthbert-dotcom/clarity-backend) to be running on `localhost:5000`.
